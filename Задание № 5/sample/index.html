Еще один вариант: для изображений добавили float:left

Обратите внимание, что при использовании инструментов разработчика, при выделении блока, изображения выходят за его границы.

Мы вывели их с помощью float из основного потока контента. При этом оставшиеся элементы блока, также как и следующий блок, их обтекают.

Раскомментируйте строчку с clear:left для текста этого блока.
