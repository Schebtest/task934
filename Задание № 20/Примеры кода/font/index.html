По умолчанию некоторые элементы формы не наследуются font-family и font-size от своих родителей.

Вместо этого многие браузеры используют внешний вид системы по умолчанию. Чтобы стиль ваших форм соответствовал остальному содержимому, вы можете добавить для них настройку стиля со значением inherit.

Форма слева с настройками по умолчанию, справа - с заданными стилями.
